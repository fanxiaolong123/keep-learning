## Step 4 — 整合：连入你的知识网络

### 为什么最后一步不是"总结"
总结是被动的——AI 帮你整理笔记不会让你记住任何东西。真正的整合是主动的：把新知识连接到你已有的知识网络中。认知科学叫 elaborative interrogation（精细加工）和 transfer（迁移）。

### 怎么做
通过对话引导用户做主动迁移：

1. **跨域连接**：
   - "这个概念让你想到你已有知识中的什么？"
   - "在你的工作/生活中，哪里能用到这个？"
   - "这个原理和你之前学过的 [X领域] 有什么相似之处？"

2. **教别人测试**（费曼技巧）：
   - 让用户假设要向一个完全不懂的人解释整个主题（不是单个概念，是整体框架）
   - 用户讲，AI 扮演"不懂的听众"提问
   - 哪里讲不清楚 = 哪里还没真正理解

3. **输出知识结构**（AI 协助）：
   - 核心模型（这个领域的底层规律是什么）
   - 关键术语速查
   - 常见误区 vs 正确理解
   - 用户个人的盲区标记（未来要补的）
   - 下一阶段学习方向

### 掌握度终判

| 维度 | 标准 |
|------|------|
| 理解 | 能向别人解释，不需要背诵 |
| 预测 | 能预测"改变变量会怎样" |
| 辨别 | 能在交错场景中正确选择和应用框架 |
| 实践 | 能独立完成对应类型的实践任务 |
| 迁移 | 能在另一个领域使用同样的原理 |

五项都达标 = 真正掌握。

### 本步产出
用户的个人知识库条目 + 明确的"下一步学什么"方向 + 学习状态快照。
